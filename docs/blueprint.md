# **App Name**: Hyderabad Reads

## Core Features:

- Book Catalog: Browse a catalog of books available for rent or purchase, filtering by genre, author, and availability within Hyderabad.
- Wishlist Management: Users can manually add or remove items from their wishlist.
- Shopping Cart: Add books to a shopping cart for purchase or rental.
- Checkout Process: A smooth and efficient checkout process with integrated delivery charge calculation for users in Hyderabad.
- Community Discussion: A community page where users can connect with friends and discuss books.
- Book Donation Platform: Donors can donate books, specifying their condition, and arrange for pickup with delivery charges applied.
- Community Discussion platform : Where the readers. buyers, donors can discuss about the books, host topics, discussions etc

## Style Guidelines:

- Primary color: Black (#000000) for a modern and sophisticated feel.
- Background color: Off-white (#F0F0F0) to provide a neutral and clean backdrop.
- Accent color: Dark gray (#333333) for contrast and emphasis.
- Body text: 'Inter', a grotesque sans-serif with a modern, machined look.
- Headline Font: 'Inter Display', a Inter with a techy and scientific feel, used for emphasis and a contemporary look.
- Use simple, minimalist icons in black and white to represent book genres, user actions, and donation options.
- A clean, grid-based layout with plenty of white space to ensure readability and a clutter-free user experience.
